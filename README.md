# Stock-scope_Dashboard

1-Project Title                                                 
   Pakistan Stock Exchange (PSX) Live Market Analysis Dashboard                                               

2-Short Description                                                                                                                            
A dynamic and interactive stock market dashboard built in Power BI to analyze live trading activity of selected stocks listed on the Pakistan Stock Exchange (PSX). The dashboard provides insights into stock performance, trading volume, gain/loss trends, historical price movement, and top-performing stocks through clear visualizations and KPI metrics. And transformed into an easy-to-understand analytical dashboard for investors and market analysis enthusiasts.
                                   
3-Tech Stack                                                                                                                
   Power BI Desktop – Dashboard development & visualization                                                                            
    Power Query – Data cleaning and transformation                                                                                             
    DAX (Data Analysis Expressions) – KPI calculations and measures                                                                            
    Excel / CSV Data Handling – Data preparation                                                                                                  
    Data Visualization Techniques – Charts, KPI cards, tables, slicers, and filters                                                           

4-Data Source                                                                                                   
   Pakistan Stock Exchange (PSX) Data                                                                 
   Source Website: scstrade.com                                                                         
 
5-Features:    
📊 Interactive Dashboard Navigation                                                                 
    • Multi-page dashboard with:  Home , Summary , Today's Stock Prices , Historical Stock Price                                                 
    • Easy page navigation using interactive buttons and filters.                                                                                    
📈 Real-Time Stock Analysis                                                   
    •Tracks:                                                      
        Today's Closing Prices                                                        
        Trading Volume                                                       
        Gain/Loss Percentage                                                                   
        Previous Close Comparison
        Provides quick insights into market movement and stock performance.
        
🏆 Top Gainers & Top Losers                                                                                              
     • Highlights best and worst-performing stocks based on daily percentage change.                                                  
     • Helps identify market trends instantly.
     
📅 Historical Trend Analysis                                                                                                             
     • Analyzes stock performance across different days and months.                                                                    
     • Displays: Opening Prices, Closing Prices, High & Low Prices ,Total Trading Volume.                                                         
     • Useful for trend forecasting and performance tracking.        
      
🔍 Stock Selection Filter                                                                          
      • Users can select individual stocks for detailed analysis.                                                                                 
      •  Dynamic visuals automatically update based on selected stock.
      
📌 KPI Summary Cards                                                             
       • Displays important stock market indicators including:  Total Volume , Previous Close ,Current Index ,Market High , Market Change                • Provides quick decision-making insights.                    
       • Trend Visualizations
       • Makes complex stock data easier to interpret and analyze  
       
📉 Visual Analytics                                                                                        
    • Includes:                                                                                           
      Line Charts                                                                                            
      Comparison Charts                                                                                                  
      Tables
